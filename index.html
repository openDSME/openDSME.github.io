Open-Source Implementation of the IEEE 802.15.4 DSME Link Layer

openDSME is meant to be portable for various platforms, including simulation environments and hardware platforms. Therefore, multiple modules are required to run the software.

openDSME

The repository of the core module can be found at github.com/openDSME/openDSME. Probably you never have to check out this repository directly, because it is integrated as a submodule in the other repositories.

INET DSME

Currently only the OMNeT++ simulation environment is supported using the INET framework. The adaptation logic to run openDSME inside of INET is implemented in the repository github.com/openDSME/inet-dsme. Simulation results from the latest test run are available at www.opendsme.org/inet-dsme.

INET

For development we use a fork of the INET framework at github.com/openDSME/inet, including minor additions to the integration branch of github.com/inet-framework/inet.
