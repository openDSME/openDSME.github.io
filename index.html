openDSME

Open-Source Implementation of the IEEE 802.15.4 DSME Link Layer

Follow me on GitHub

openDSME is meant to be portable for various platforms, including simulation environments and hardware platforms. However, currently only the OMNeT++ simulation environment is supported.

OMNeT++ / INET

The current development of openDSME takes place inside a fork of the INET framework that can be found at https://github.com/openDSME/inet.
